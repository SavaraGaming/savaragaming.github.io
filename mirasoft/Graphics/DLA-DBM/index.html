DLA and DBM

Controls:
1-9: Input parameter sets
Space: Toggle simulating
T: Disable rendering for turbo speed

Concept:
This project simulates the growth branching structures using two different algorithms: diffusion limited aggregation (DLA) and dielectric breakdown model (DBM). Each algorithm decides where to add growing material differently, but both are able to modify the thickness of branches. Thickness can range from single pixel branches to a large clumping mass with no distinct branches. As an added gimmick, branching material can be preset in the environment for material to grow from.
