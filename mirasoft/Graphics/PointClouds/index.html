Point Clouds

Concept:
This is an example of generating 3D point clouds with OpenSFM (Open source Structure From Motion pipeline). The input image set is a set of ~50 images rotating perspective about the main entrance of the Library. I made several of these, and I thought this one was the most richly detailed.
